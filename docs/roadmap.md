# PhantomScope Roadmap

## MVP

- Accept a target brand or domain
- Generate suspicious domain variants
- Query CT data with offline fallback
- Enrich with RDAP, DNS, IP, ASN, and basic reputation
- Apply explainable risk rules
- Produce analyst-oriented summary text
- Export Markdown and JSON reports
- Run via FastAPI and Streamlit

## V2

- Async job queue and scheduled re-scans
- Historical comparisons for newly observed domains
- Passive DNS and screenshot enrichment adapters
- More nuanced infrastructure clustering
- Postgres storage layer
- Alerting webhooks and email digests

## V3

- Investigation workbench and evidence graph
- Multi-tenant teams and RBAC
- Feedback-driven scoring calibration
- Case management and ticketing sync
- Provider health telemetry and rate-limit dashboards
