# Demo Flow

## Ideal 60-90 Second Walkthrough

1. Start with the dashboard home screen already open.
2. Type `acme` into the target field.
3. Keep offline mode enabled for deterministic output.
4. Run the analysis.
5. Pause on the overview metrics and analyst headline.
6. Open the highest-ranked finding.
7. Highlight:
   score
   technique
   CT observation
   registrar / ASN context
   named risk signals
8. Scroll to the narrative/export section.
9. Download the Markdown report.
10. End on the idea that PhantomScope helps analysts triage suspicious external infrastructure with explainable logic.

## Recording Tips

- Keep the cursor movement deliberate and slow.
- Use one consistent browser width.
- Avoid changing targets mid-recording.
- Do not spend too long on setup screens.
- Show at least one `MOCK` provenance badge so viewers understand the demo is honest and reproducible.

## Suggested Caption

PhantomScope analyzes suspicious lookalike domains, enriches surrounding infrastructure, applies explainable risk scoring, and produces analyst-ready summaries in a portfolio-safe defensive workflow.
