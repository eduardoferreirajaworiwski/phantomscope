Capture tips:

- Run `make demo-interview` so the visuals are stable.
- Use offline mode to avoid changing live-provider output during recording.
- Prefer one consistent target such as `acme`.
- Capture at desktop width and ensure at least one frame shows `MOCK` provenance and score rationale.
